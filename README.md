qal/flickr: Flickr web service client for PHP (ALPHA)
=====================================================

**qal/flickr** is a service for interacting with the Flickr API based on [Guzzle][1].

[OAuth authentication][2] is supported.

## Requirements

* PHP 5.3.3 or later
* [OAuth extension][3]
* [Guzzle][1] 2.6 or later
